# Monstrous_Creature_Game_Translation
基于Frida-il2cpp-bridge的dmm妖怪剧情文本翻译。<br>
仅翻译剧情文本。<br>
仅支持安卓版。<br>
<br>
使用：<br>
下载安装包和Translation/fridaTranslation.json。<br>
启动游戏后退出，将fridaTranslation.json复制到Android/data/jp.co.dmm.dmmgames.ayarabux.luojun（沙盒文件夹）中。<br>
然后再次启动即可。<br>
<br>
注意：<br>
因为是本地翻译所有翻译文件均在json文件中所以新文本会没有翻译。
